# Eenvoudige Console Applicatie: Contactenlijst

Ontwikkel een eenvoudige console applicatie in C# die gebruikers in staat stelt om contacten toe te voegen, te bekijken en te verwijderen. De applicatie moet de contactgegevens opslaan in een lijst binnen het geheugen. Dit project is bedoeld om basiskennis van C# en objectgeoriënteerd programmeren te testen.

## Opdracht

Implementeer de gevraagde functionaliteit.
